ENDPOINTS

¿Cuál es el camino?

Un "endpoint" es la URL, que a menudo incluye la URL y los parámetros de consulta, a la que envía una solicitud para recibir los datos que está buscando. Los puntos finales se organizan en categorías amplias según el tipo de información que proporcionan. En este caso al ser un API pequeño, solo se cuenta con 4 URLs.

LAS PETICIONES DE LAS QUE DISPONE

  • GET RECIPES

    • Con esta petición se obtiene el listado de recetas.

  • GET RECIPE

    • Con esta petición se obtiene una receta por su id.

  • POST RECIPE

    • Con esta petición se agrega una receta.

  • PUT RECIPE

    • Esta es la petición para actualizar la información de una receta.

  • DELETE RECIPE

    • Con esta petición se elimina una receta
